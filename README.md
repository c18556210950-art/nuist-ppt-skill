# 南京信息工程大学 PPT 生成器

基于 NUIST 官方 PPT 模板，用 Python 快速生成学术演示文稿。

## 快速开始

```bash
pip install python-pptx
python generate_ppt.py
```

修改 `generate_ppt.py` 顶部配置区的内容，运行即得 `output.pptx`。

## 文件说明

| 文件 | 说明 |
|------|------|
| `SKILL.md` | 技能定义与完整文档 |
| `generate_ppt.py` | PPT 生成脚本（修改配置→运行→输出） |
| `南京信息工程大学ppt模版.pptx` | 官方模板（只读，勿改） |
| `test_10_sections.py` | 10 章节测试脚本 |
| `test_limits.py` | 极限字数测试脚本（25 页） |

## 版式

模板内置 11 种版式：封面、目录、节标题、标题和内容、三段式、图文对比、自我介绍、仅标题、空白、结尾。

## 字数限制

| 版式 | 字段 | 上限 |
|------|------|------|
| 封面 | 标题/副标题 | 15 字 |
| 目录 | 每项 | 15 字 |
| 节标题 | 标题 | 12 字 |
| 内容页 | 标题 | 20 字 |
| 内容页 | 每条要点 | 35 字 |
